# 여러 경고들에 대해 공부하기

## ``Unnecessary @Autowired annotation` (VSCode Extension)

* 생성자가 하나라면, Spring Boot는 알아서 @Autowire한다
* 생성자가 여러 개일 때 무엇에 Autowire할지 지정할 필요가 있다
* [관련 문서](https://docs.spring.io/spring-framework/reference/core/beans/annotation-config/autowired.html)

* 그래도 명시적으로 적어주는게 좋지 않을까? -> 그렇다면 이 경고를 꺼주는 것이 좋다
